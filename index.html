Overview

Bgolearn is an open-source platform designed to enhance material design through advanced Bayesian optimization techniques. With powerful tools for regression, classification, and multi-objective optimization, Bgolearn empowers researchers and engineers to efficiently explore material properties and design parameters.

Key Components

Bgolearn

The core module offering powerful regression, classification, and efficiency testing tools. Optimized for material property prediction and analysis.

MultiBgolearn

Focuses on multi-objective Bayesian optimization. Enables the simultaneous optimization of multiple objectives, with flexible surrogate model selection.

BgoFace

A user-friendly graphical interface for interacting with Bgolearn. It simplifies the process of running optimization tasks and visualizing results.

Installation

To install Bgolearn and its modules, run the following commands:

pip install Bgolearn
pip install MultiBgolearn
